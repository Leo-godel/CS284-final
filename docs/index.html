Final Project Proposal: Snow Simulation With G2P2G and Optimization

Summary

Efficient snow simulation is a challenging task. In this project, we aim to implement efficient parallel MPM snow simulation algorithm that can run quickly on GPU platforms. We will use our algorithm to simulate complex scenes as well as make comparison and benchmarking between optimized algorithm and unoptimized versions.

Problem Description

Snow simulation is a challenging task due to the complex physical nature of snow. Material Point Method (MPM) has been widely used in snow simulation to produce realistic simulation effects. However, MPM is a computational heavy and time consuming algorithm. The state-of-the-art optimized CPU algorithm takes more than over 500 seconds in simulating a complex scene including 5M particles. In recent years, more research dives into how to effectively utilize GPU to speed up the MPM algorithm. A recent work published in SIGGRAPH 2020 reformulated the MPM algorithm's pipeline and designed new Array-of-Structs-of-Array (AoSoA) data structure to further speed up the MPM performance.

Goals and Deliverables

Our goal in this project is to use re-structured MPM algorithm with a fused G2P2G kernel function and AoSoA data structure to speed up the snow simulation process.

Our minimum final deliverable will include a few demonstration of final snow simulation results as will as performance comparision between optimized GPU algorithm, a few optimization milestones, and unoptimized CPU algorithm. We hope that we at least achieve 10x speed up with the help of state-of-the-art GPU algorithm.

If things go really smooth, we will have more time spent on the optimization stage, so that we may try more optimization methods and have higher performance improvement.
